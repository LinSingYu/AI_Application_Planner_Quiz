# AI 應用規劃師 (初級) 模擬測驗系統  
本專案旨在提供 經濟部產業人才能力鑑定 (iPAS) - AI 應用規劃師 (初級) 的互動式模擬測驗平台。透過 GitHub Pages 部署，讓考生能隨時隨地進行測驗，掌握「生成式 AI 應用與規劃」的核心職能

🔗 立即開始測驗
https://linsingyu.github.io/AI_Application_Planner_Quiz/

本測驗題目根據官方學習指引「科目一、科目二」進行編制 ，重點包括：

- 人工智慧
  - AI的定義與分類
  - AI治理概念
- 資料分析與處理
  - 資料基本概念與來源
  - 資料整理與分析流程
  - 資料隱私與安全
- 大數據分析
- 機器學習主題
- 深度學習
- 自然語言處理
- 生成式AI(GAI) + 鑑別式AI(DAI)

### 科目二
- No Code
- Low Code
- 生成式AI工具
- 生成式AI技術應用
- 生成式AI導入應用評估規劃
- AI治理
- AI風險管理

## 🚀 功能特色
互動式介面：使用 Google Gemini 協助製作，利用 HTML/JavaScript 打造，即時回饋答案正誤。
詳細解析：每題測驗皆附帶考題解析，幫助學習者理解題目背後的知識點 。
免安裝：直接透過瀏覽器開啟即可開始測驗，支援電腦與手機瀏覽。
開源共享：歡迎社群共同維護題庫，增加題目多樣性。
